## Project Overview

Simon Says is an interactive memory game where the player must remember and repeat a sequence of colored buttons generated by the system. With each successful level, a new color is added to the sequence, increasing the difficulty and testing the player's memory and concentration.

## How It Works

The game maintains two arrays:

* **gameSeq** → Stores the sequence generated by the system.
* **userSeq** → Stores the sequence entered by the player.

### Game Flow

1. The player starts the game using the Start button.
2. The system generates a random color and adds it to the game sequence.
3. The player must reproduce the sequence by clicking the colored buttons in the correct order.
4. After each click, the player's input is compared with the corresponding value in the game sequence.
5. If the entire sequence is correct, the game advances to the next level.
6. If an incorrect button is clicked, the game ends and the score is displayed.
