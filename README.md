# Simon Says Game

A memory-based web game built using HTML, CSS, and JavaScript.

## Project Overview

Simon Says is a memory game where the player must remember and repeat a sequence of colored buttons generated by the system. With each level, a new color is added to the sequence, making the game progressively more challenging.

## Features

* Dynamic sequence generation
* Level progression system
* High score tracking using Local Storage
* Mobile responsive design
* Start button for improved user experience
* Interactive button animations
* Prevention of consecutive duplicate colors

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

The game maintains two arrays: one for the sequence generated by the system and another for the sequence entered by the user.

At each level, a random color is added to the game sequence. The player must reproduce the sequence by clicking the colored buttons in the correct order.

After every click, the player's input is compared with the corresponding element in the game sequence. If all inputs match, the game advances to the next level. Otherwise, the game ends and the score is displayed.
